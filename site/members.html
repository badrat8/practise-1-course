Работа над реализацией сервиса визуализации данных колл-центра

Для реализации была разработана архитектура (представлена на схеме). Вуз использует систему Asterisk в качестве автоматической телефонной станции, данные из нее собираются в базу данных при помощи стороннего приложения и подключаются к сервису Grafana для дальнейшей визуализации.

Фрагмент скрипта, который подключается к AMI Asterisk, на языке Python. Данный код забирает данные из телефонии и запускает их обработчик. Сохраняются имя очереди, количество звонков (принятых и отклонённых), время удержания, звонка и так далее. После чего эти данные записываются в строки базы данных.

Также мне удалось попрактиковаться в написании sql запросов в Grafana.

На данный момент реализовано только два параметра, впереди реализация еще, как минимум, десяти. Система ещё не готова к работе, но мы проводим тесты. В будущем также планируется разработка полноценной промежуточной среды для сбора данных.
